Note Explicative sur l'Usage de l'Intelligence Artificielle (IA)

Projet : Portfolio M2 Management du Sport - Organisation du Tournoi de Pickleball

1. Outil Génératif Utilisé

L'intégralité de la base de code (structure HTML, CSS, et premier jet du contenu textuel) a été générée en utilisant l'outil conversationnel Gemini (Google).

2. Rôle de l'IA dans le Développement

L'IA a été sollicitée pour :

Initialisation du Projet : Création d'une application monopage (SPA) en HTML/Tailwind pour répondre aux premières contraintes.

Structuration Multi-Fichiers : Transformation de la SPA en architecture multi-pages (5 fichiers HTML + 1 fichier CSS) avec gestion des liens et des chemins relatifs.

Génération de Contenu : Rédaction initiale des blocs de texte pour les pages (Présentation, Événement, Programme, etc.) basés sur les informations fournies par l'étudiant (Alexandre COEURET, M2 Nanterre, Pickleball, Sports Extrêmes).

Définition du CSS : Traduction des classes Tailwind en règles CSS pures et responsives dans le fichier style.css.

3. Modifications Manuelles et Personnalisation (Reformulation et Corrections)

Conformément à la consigne 4, les modifications manuelles suivantes ont été appliquées pour personnaliser et améliorer le rendu :

Réorganisation des Sections : La navigation et la distribution du contenu ont été ajustées manuellement pour assurer une séparation claire sur les 5 pages, notamment la fusion du profil/formation/université sur la première page.

Ajout de Détails dans la Navigation : Le menu de navigation dans chaque fichier HTML a été modifié pour utiliser des libellés précis (ex: "1. Présentation Générale", "5. Inscriptions") et assurer que les chemins d'accès pointent correctement vers les .html adjacents.

Reformulation du Titre de l'Événement : Le titre du tournoi a été fixé à "PickleRévolution" sur toutes les pages pour une meilleure cohérence.

Structure et Sémantique HTML : Les balises HTML ont été simplifiées (retrait des classes Tailwind inutiles) et ajustées pour coller aux nouvelles règles CSS.

L'utilisation de l'IA a permis d'accélérer le développement initial, tandis que les modifications manuelles garantissent la conformité aux contraintes techniques et la personnalisation du contenu.
